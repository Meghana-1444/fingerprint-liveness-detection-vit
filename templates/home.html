About This Project

Fingerprint Liveness Detection (FLD) is essential for secure biometric authentication, protecting systems from spoofing attacks made using silicone, gelatine, or latex fingerprints. Traditional CNN and multimodal methods work well but often add complexity and high computational cost.

Our project introduces a lightweight deep learning framework that detects subtle fingerprint cues such as ridge distortions, pore distribution, and perspiration patterns using an efficient detection model with an enhanced backbone and decoupled head.

Unlike multimodal systems, our model operates on fingerprint images alone, achieving high spoof detection accuracy, faster inference, and excellent generalization against new attack materials — making it practical for real-world biometric security systems. 🔐
